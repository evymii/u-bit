# u-bit

Monorepo for u-bit application.

## Structure

```
u-bit/
├── apps/
│   ├── backend/          # Backend API
│   └── frontend/         # Next.js frontend
├── packages/
│   └── shared/           # Shared types and utilities
└── docs/                 # Project documentation
```

## Getting Started

### Backend

```bash
cd apps/backend
npm install
npm run dev
```

### Frontend

```bash
cd apps/frontend
npm install
npm run dev
```
